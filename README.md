# Javafx

## Runtimes
 - JavaFx platform
 - Gluon mobile application framework

## UI layouts
 + StackPane
 + AnchorPane
   - As the name implies it anchors the child nodes to the pane
   - It lets us to position nodes relative to the pane edges   
 + GridPane
 + FlowPane and TilePane
 + BorderPane
 + SplitPane
 + HBox, VBox, and ButtonBar

## Tools
 - Scene builder
